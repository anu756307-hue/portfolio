Site Report-csy1063 Web Development

This portfolio website is my project of csy1063 Web Development module. this assignment helped me understand the structure of a website using only html,css , along with responsive design and using github.i learned how different web pages link together and how to organize a website effectively . the purpose of this website is to design and build a responsive portfolio website that showcases my projects skills in Web Development .

During the development of the website, I learend how to structure web pages using semantic html elements such as header, nav,section, and footer. Using the elements improve the readability and structure of my code. I also gained experience in styling pages using css , coloring , padding , position ,color and learned how to manage layouts using CSS Grid column .

One of the challenges I faced was creationg a responsive layout ,creating hamburger icon for mobile screens. i initially struggled with positioning elements correctly , especially when adjusting the the layout. And also had to difficulty understanding the values bettwen pixels and percentages in layouts. after trials and testing , I was able to sucessfully had layout abd hide the hamburger icon.

For the design , i choose a simple layout and colour scheme to make the website easy to read and navigate. I avoided using complex colours , focused on usability and minimal design . To ensure mt code , I validated my codes using online validators(w3c). Any errors found were corrected to ensure the code met web standards. Screenshots of the validation results are included below.

Artificial Intelligence tools were used for guidance, helping me understand CSS layouts and for debugging code errors. Overall ,this project enhanced my understanding of web development , taught me how to solve problems during coding , and helped me gain confidence in creating professional-looking web pages.
